# All-About-Electronics
This repository archives my hardware and electronics engineering projects. It includes everything from low-level transistor logic to integrated sensor systems, providing a window into my design-to-build workflow.

⚡ Project Categories

Circuit Design: Analog/Digital schematics and circuit analysis.

PCB Design: Custom layouts, board fabrication files, and signal routing.

Prototyping: Sensor integration, radio transmission, and hardware testing.

Digital Logic: Transistor-level designs and RTL implementations.

🛠 Tech Stack

Design Tools: KiCad, Multimeter, Oscilloscope

Circuit Analysis: KiCad, oscilloscope

Logic Theory: Resistor-Transistor Logic (RTL), Transistor Transistor Logic (TTL), CMOS, combinational logic, IC logic

📁 Repository Structure

/schematics: Circuit diagrams and functional blocks.

/pcb-files: Board layouts and manufacturing files.

/prototypes: High-resolution photographs, test results, and build documentation of physical hardware implementations.
